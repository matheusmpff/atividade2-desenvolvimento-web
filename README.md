# atividade2-desenvolvimento-web
## Descrição
A atividade proposta pela disciplina SCC0219 - Introdução ao Desenvolvimento Web (2026) da USP, consiste em desenvolver um Web Service para Annabelle Merigold que possui uma loja de poções bem-sucedida, localizada no Beco da Última Saída, chamada “Poções e Soluções". A aplicação Web deve permitir a vendedora cadastrar, listar e remover poções, com cada uma contendo os campos nome, descrição, imagem e preço. Para isso será necessário uma página de administração que permita realizar as ações comentadas, bem como uma página para o comprador.

## Página do Comprador
Nessa página, o exercício propõe que certas informações estejam presentes, sendo elas:
+ Descrição da loja
+ seção de histórico da loja
+ Rodapé com informações de contato
+ Toda poção precisa ter me, imagem, descrição, preço e um botão “Comprar”(Não é preciso a implementação da compra nessa entrega)

## Informações sobre a cliente
A cliente por sua vez, demonstra interesse em fontes clássicas e sombrias de forma a orientar o desenvolvedor a necessidade de seguir paleta de cores que irá compor o site de forma a serem mais escuras. 

## Execução 
Instale as dependências em cada pasta que contem um package.json executando o comando npm install. Após isso, execute npm run dev em cada pasta para executar os programas de backend front end para o usuário e a página de administração. No total há 3 pastas dentro delas tem o projeto. Acesse o projeto e execute as instruções a cima. O back-end ja inclui alguns produtos no banco previamente para visualização de como ficaria nas páginas.
